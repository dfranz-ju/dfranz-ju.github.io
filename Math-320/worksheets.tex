\typeout{Subsection 1.1.1 Solving Systems of Linear Equations}
\typeout{************************************************}
%
\begin{subsectionptx}{Solving Systems of Linear Equations}{}{Solving Systems of Linear Equations}{}{}{x:subsection:subsection-systems-basics}
A system of linear equations has%
\begin{itemize}[label=\textbullet]
\item{}no solution, or%
\item{}exactly one solution, or%
\item{}infinitely many solutions.%
\end{itemize}
%
\par
We solve linear systems of equations by putting them into augmented matrix form and then simplifying them using one or more of the following \terminology{elementary row operations}:%
\begin{description}
\item[{Replacement:}] Replace one row by the sum of itself and a multiple of another row.\item[{Interchange:}] Interchange two rows.\item[{Scaling:}] Multiply all entries of a row by a nonzero constant.\end{description}
%
\end{subsectionptx}

\typeout{Subsection 1.1.3 Using Row Operations in Sage}
\typeout{************************************************}
%
\begin{subsectionptx}{Using Row Operations in Sage}{}{Using Row Operations in Sage}{}{}{g:subsection:idm181732458448}
Let's investigate how to use Sage to "simplify" augmented matrices using the three row operations.%
\begin{sageinput}
A=matrix(QQ, [[5,3,1], [1,4,-2]]); A
\end{sageinput}
To use the Interchange row operation, we use the \mono{swap\_rows} method. Note that Sage starts counting at 0, so the first row is row 0.%
\begin{sageinput}
A.swap_rows(0,1); A
\end{sageinput}
\begin{sageoutput}
[1 4 -2]
[5 3 1 ]
\end{sageoutput}
We now use a Replacement row operation; since this operation adds a multiple of one row to another row, the method name is \mono{add\_multiple\_of\_row}.%
\begin{sageinput}
A.add_multiple_of_row(1,0,-5); A
\end{sageinput}
To understand the syntax of the method \mono{add\_multiple\_of\_row}, read it right to left. In the above example, we added \(-5\cdot\) Row 0 to Row 1 (where Row 0 is the first row).%
\par
A Scaling operation will make that second row look nicer. We use the \mono{rescale\_row} method.%
\begin{sageinput}
A.rescale_row(1, -1/17); A
\end{sageinput}
