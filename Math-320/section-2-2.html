Section 2.2 Guided Practice 2.2: The Inverse of a Matrix

Overview.

Now that we understand how to add and multiply matrices, we can think a little deeper about these operations. In particular, since we are interested in solving matrix equations of the form \(A\mathbf{x}=\mathbf{b}\text{,}\) we want to explore matrix multiplication a little more.

Remembering that matrix equations are generalizations of simple linear equations, we look to a simple linear equation as a motivating example. Consider the equation \(5x=3\text{.}\) To solve it, we multiply both sides by \(1/5\text{,}\) the multiplicative inverse of \(5\text{,}\) because \(\dfrac{1}{5}\cdot 5=1\text{.}\) This yields \(x=\dfrac{1}{5}\cdot 3.\)

To accomplish something similar for the matrix equation \(A\mathbf{x}=\mathbf{b}\text{,}\) we need a matrix \(C\) such that \(CA=I\text{,}\) where \(I\) is an appropriately sized identity matrix. We call such a matrix an \(inverse\) matrix. This section explores the properties of inverse matrices, how to determine if they exist, and how to find them if they do.

Subsection 2.2.1 Basic Learning Objectives

You should be able to perform these tasks with reasonable fluency when you arrive to class.

  • State what it means for a square matrix to be invertible.

  • Compute the determinant of a \(2\times 2\) matrix and use it to determine if the matrix is invertible.

  • Find the entries of the inverse of an invertible \(2\times 2\) matrix.

Subsection 2.2.2 Advanced Learning Objectives

In addition to mastering the basic objectives, here are the tasks you should be able to perform after class, with practice:

  • Correctly use the properties of inverse matrices in Theorems 5 and 6.

  • Use row reduction to determine if a square matrix is invertible.

  • Use row reduction to find the inverse of a an invertible matrix.
