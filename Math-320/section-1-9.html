Section 1.9 Guided Practice 1.9: The Matrix of a Linear Transformation

Overview.

Because we have already done so much work with matrices, it would be nice to put that knowledge to work when it comes to linear transformations. That is, we would like to be able to describe all linear transformations as matrix transformations. As Example 1 in the section demonstrates, this is actually something we can always do! (At least as long as the linear transformation is mapping from \(\mathbb{R}^m\) to \(\mathbb{R}^n\))

After Example 1, the book discusses the general strategy for finding the standard matrix of a linear transformation; in other words, the matrix formula for the transformation. We then see the standard matrices for many geometric transformations of the plane, including rotations, reflections, scaling, shears, and projections. Some of these will be explored in more detail in an upcoming project on computer graphics.

The section concludes by discussing the properties of a function being one-to-one or onto. Once we have a matrix formula, these properties directly correspond to statements about the columns of the matrix being linear independent or spanning \(\mathbb{R}^m\text{;}\) see Theorem 12. Linear transformations will continue to be an important topic, and this section gives us some of the main tools to help analyze them.

Subsection 1.9.1 Basic Learning Objectives

You should be able to perform these tasks with reasonable fluency when you arrive to class.

  • Explain how to find the standard matrix of a linear transformation \(T: \mathbb{R}^n \to \mathbb{R}^m\text{.}\)
  • Given a linear transformation \(T: \mathbb{R}^n \to \mathbb{R}^m\text{,}\) find its standard matrix.
  • Find the standard matrix of the linear transformation that rotates each point of \(\mathbb{R}^2\) counterclockwise by \(\theta\) radians.

Subsection 1.9.2 Advanced Learning Objectives

In addition to mastering the basic objectives, here are the tasks you should be able to perform after class, with practice:

  • Give geometric descriptions of the matrix transformations in Tables 1-4.
  • Explain what it means for a linear transformation to be one-to-one or onto, without referencing a matrix.
  • Use the standard matrix of a linear transformation \(T\) to determine if \(T\) is one-to-one or onto.
