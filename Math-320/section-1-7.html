Section 1.7 Guided Practice 1.7: Linear Independence

Overview.

So far we have discussed systems of linear equations, including writing them as vector equations and matrix equations, linear combinations and spans, and homogeneous systems. Conceptually, linear independence, the focus of this section, relates to all these topics.

One way to think about linear independence is to compare and contrast with the idea of spans. With spans, we are focused on which vectors we can create from the building blocks we start with (i.e. what vectors can we write as linear combinations of the starting vectors). Think of the starting vectors as types of LEGO pieces, and the Span of those vectors as containing all LEGO creations you could build with those types of pieces. The notion of linear independence addresses the question "did we need all those starting types of LEGO pieces, or could we have started with fewer and still build everything?"

In other words, linear independence is concerned with whether or not there is extra, redundant, information present. If a set of vectors is linearly independent, there is no redundant information; everyone one of those vectors is needed to build the Span. If the vectors are linearly dependent, then there is redundant information; we could build the same Span while getting rid of some of the starting vectors.

While that is a big picture introduction, the text explores linear independence from the perspective of matrix and vector equations (including the actual definition of linear independence and dependence) and relates it to geometry. It will be helpful to read this section multiple times while keeping the conceptual description above in mind; linear independence can be a tricky idea to master, but it will play a central role in many topics going forward.

Subsection 1.7.1 Basic Learning Objectives

You should be able to perform these tasks with reasonable fluency when you arrive to class.

  • State what it means for a given set of vectors to be linearly independent.
  • State what it means for a given set of vectors to be linearly dependent (without referencing linear independence).
  • Quickly determine if a set of 2 vectors is linearly independent with minimal computation.

Subsection 1.7.2 Advanced Learning Objectives

In addition to mastering the basic objectives, here are the tasks you should be able to perform after class, with practice:

  • Determine if a given set of vectors is linearly independent.
  • Find linear dependence relations among linearly dependent vectors.
  • Apply Theorems 8 and 9 to appropriate sets of vectors to determine linear dependence.

Subsection 1.7.4 Sage Commands

Continuing a trend, there are no new commands to learn. Our primary computations are still row reducing an appropriate matrix to answer our questions. Computationally deciding if a set of given vectors is linearly independent involves creating a matrix with the vectors as columns and then row reducing it, which we already know how to do using the column_matrix command and .rref() method. The interpretation is then up to us.
