Section 1.8 Guided Practice 1.8: Introduction to Linear Transformations

Overview.

The matrix equation \(A\mathbf{x}=\mathbf{b}\) was introduced in Section 1.4 as another way to notationally represent a vector equation, or system of linear equations, with \(A\) representing the coefficient matrix of the system. But the quantity \(A\mathbf{x}\) arises naturally in other contexts, such as computer graphics or dynamical systems. In these contexts we think of the matrix \(A\) as an object that "acts" on a vector \(x\) to produce a new vector \(A\mathbf{x}\text{.}\)

Mathematically, we describe such actions as functions, and we call the functions that arise from changing one vector into another vector transformations. Fortunately, while this is a completely differenent perspective on \(A\mathbf{x}\text{,}\) and as a result the kinds of questions we ask are now different, we can still use the tools of row reduction and pivots to answer these questions by rephrasing them in terms of systems of equations or matrix equations. Example 1 of this section is a great illustration of how these ideas all combine together.

While there are many possible kinds of transformations, in this class we will study linear transformations (as the name of the course itself might imply), which are formally defined roughly halfway through the section. Linear transformations arise naturally in many applications, and all transformations defined by multiplication by a matrix are linear transformations (we will explore the other direction of this in the next section). The properties of linear transformations make them particularly nice to analyze, and we will take full advantage of that fact.

Subsection 1.8.1 Basic Learning Objectives

You should be able to perform these tasks with reasonable fluency when you arrive to class.

  • Describe what transformations and matrix transformations are.
  • Identify the domain and codomain of a given matrix transformation.
  • Given a transformation \(T\text{,}\) compute the image of a specific vector under the action of \(T\text{.}\)
  • Define what it means for a transformation to be a linear transformation.

Subsection 1.8.2 Advanced Learning Objectives

In addition to mastering the basic objectives, here are the tasks you should be able to perform after class, with practice:

  • Given a transformation \(T\text{,}\) determine if a given vector is in the range of \(T\text{.}\)
  • Determine if a given transformation is a linear transformation.
  • Geometrically describe the actions of projection, shear, and scaling matrix transformations.
