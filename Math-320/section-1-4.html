Section 1.4 Guided Practice 1.4: The Matrix Equation \(A\mathbf{x}=\mathbf{b}\)

Overview.

In Section 1.3, we saw that we could reframe a system of linear equations as a single vector equation, in which one side was a linear combination of vectors. In this section, we continue to reframe systems of equations, this time by replacing a linear combination of vectors by a product of a matrix and a vector (see the definition on page 35). This creates a matrix equation which is equivalent to the vector equation and system of linear equations.

A major benefit of using matrix equations is that it helps us more easily analyze properties of linear systems. The discussion of this in the textbook culminates in Theorem 4, which relates consistency, linear combinations, spans, and pivot positions of a relevant matrix. This theorem is one of our main tools in answering problems and translating between different interpretations and perspectives.

The remainder of the section focuses on properties of matrix-vector multiplication. Even though we will be doing most of our multiplication with technology, it is important to be able to do small matrix-vector computations by hand, so certainly read pages 38-39 and practice the "row-vector" rule. The top of page 39 also introduces the identity matrix, an important kind of matrix we will be seeing again.

Subsection 1.4.1 Basic Learning Objectives

You should be able to perform these tasks with reasonable fluency when you arrive to class.

  • State what it means for a set of vectors to span \(\mathbb{R}^n\)
  • Compute the product of a matrix \(A\) and a vector as a linear combination of the columns of \(A\text{.}\)
  • Translate between vector equations and matrix equations.
  • Compute the product of a matrix and a vector using the row-column rule.

Subsection 1.4.2 Advanced Learning Objectives

In addition to mastering the basic objectives, here are the tasks you should be able to perform after class, with practice:

  • Determine if a given matrix equation \(A\mathbf{x}=\mathbf{b}\) has a solution for every \(\mathbf{b}\text{.}\)
  • Determine if a set of vectors spans \(\mathbf{R}^m\text{.}\)
  • Explain why the statements in Theorem 4 are equivalent.

Subsection 1.4.4 Sage Commands

Sage naturally understands multiplication of a vector by a matrix using the usual multiplication symbol (shift+8 on most keyboards). (Remember that Sage displays vectors in row form with parentheses, but correctly treats them as column vectors)

Note that the dimensions have to match up (the number of columns of the matrix should match the number of entries in the vector); if they do not, then Sage will output an error (which you can see by executing the code below).

As a reminder, if vectors are already defined, you can create a matrix using them. To make a matrix with the vectors as columns, use the column_matrix command, shown below. To make a matrix with the vectors as rows, just use the matrix command with the vectors given inside brackets.
