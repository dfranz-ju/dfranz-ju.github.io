You should be able to perform these tasks with reasonable fluency when you arrive to class.

  • Compute sums and scalar multiples of column vectors in \(\mathbb{R}^n\text{.}\)
  • List examples of linear combinations of a given set of vectors, specifying weights.
  • Translate among vector equations, linear systems, and augmented matrices.
  • Define the span of a set of vectors and produce elements of the span of a given set of vectors.
