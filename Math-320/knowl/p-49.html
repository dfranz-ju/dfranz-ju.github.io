In Section 1.3, we saw that we could reframe a system of linear equations as a single vector equation, in which one side was a linear combination of vectors. In this section, we continue to reframe systems of equations, this time by replacing a linear combination of vectors by a product of a matrix and a vector (see the definition on page 35). This creates a matrix equation which is equivalent to the vector equation and system of linear equations.
