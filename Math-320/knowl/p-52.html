You should be able to perform these tasks with reasonable fluency when you arrive to class.

  • State what it means for a set of vectors to span \(\mathbb{R}^n\)

  • Compute the product of a matrix \(A\) and a vector as a linear combination of the columns of \(A\text{.}\)

  • Translate between vector equations and matrix equations.

  • Compute the product of a matrix and a vector using the row-column rule.
