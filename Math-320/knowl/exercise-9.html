Exercise 4.
In each part, use a Sage command to apply the given elementary row operation to the matrix \(B = \begin{bmatrix} 1 \amp 2 \amp 3 \\ 4 \amp 5 \amp 6 \\ 7 \amp 8 \amp 9 \end{bmatrix}\text{.}\) Each answer should be written in a code cell that executes correctly. Note: each row operation should be applied to the initial matrix \(B\text{;}\) they should not be applied sequentially.
  1. Swap the second and third row of B.
  2. Multiply the first row of B by 4.
  3. Add the first row to the third row.
