Section 1.10 Sage commands

SageMath is a Computer Algebra System (CAS) that is both free and usable in a browser. We will be using it to perform computations and produce relevant images. The subsections below explain the commands and gives examples of how to use them.

Subsection 1.10.2 Creating line-plot images

For the computer graphics project, you will be transforming an initial image. This subsection explains how to create an image in Sage by "connecting the dots." We tell Sage the "dots" via a \(2\times n\) data matrix, where each column corresponds to a point: the top entry is the \(x\)-coordinate and the bottom entry is the \(y\)-coordinate.

While the matrix is convenient for use, Sage prefers a list of points. For this, we use the columns method on the matrix to create a list of points. Then we use the list_plot command to plot the points and connect them with line segments (reading left to right in the matrix).

The option plotjoined=true is what actually connects the dots, and the option aspect_ratio=1 forces each axis to use the same scale. You can hide the axes using the option axes=False.

Subsection 1.10.3 Transforming images

We can transform an image created as in the previous subsection by multiplying the data matrix with our transformation matrix. This has the effect of moving each point to a (possibly) new location. The lines follow a long, creating a new image.
