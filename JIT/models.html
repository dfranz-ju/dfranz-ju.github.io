Section 1.9 Linear and Exponential Models

Checkpoint 1.9.1.

Determine if the table below could represent a linear or exponential function. Then find a formula for the function the table describes.

\(x\) 0 1 2 3
\(f(x)\) 5 7 9.8 13.72

The table describes a(n)

  • Linear

  • Exponential

function with formula

\(f(x) =\)

Answer 1.

\(\text{Exponential}\)

Answer 2.

\(5\cdot 1.4^{x}\)

Solution.

There is not a constant difference between function values, so we try multplication instead. After computing \({7}/5=1.4\text{,}\) we check that it is the consistent multiple between function values:

\begin{equation*} {7} * 1.4 = {9.8} \end{equation*}
\begin{equation*} {9.8} * 1.4 = {13.72} \end{equation*}

So we conclude that this table could represent an exponential function. The initial value is \(f(0)=5\) and we found \(b=1.4\text{,}\) so \(f(x)={5\cdot 1.4^{x}}\text{.}\)

Checkpoint 1.9.2.

Determine if the table below could represent a linear or exponential function. Then find a formula for the function the table describes.

\(x\) 0 1 2 3
\(f(x)\) 8 13 18 23

The table describes a(n)

  • Linear

  • Exponential

function with formula

\(f(x) =\)

Answer 1.

\(\text{Linear}\)

Answer 2.

\(8+5x\)

Solution.

We observe that there is a constant difference between the function values of \(5\text{:}\)

\begin{equation*} {13} - 8 = 5 \end{equation*}
\begin{equation*} {18} - {13} = 5 \end{equation*}
\begin{equation*} {23} - {18} = 5 \end{equation*}

That means this table could represent a linear function. To find the formula, we note that the slope is \(5\) and the vertical intercept is \(f(0)=8\text{,}\) so the formula is \(f(x)={8+5x}\text{.}\)

Checkpoint 1.9.3.

Find the formula of a linear function whose graph passes through the points \((-4, {-7})\) and \((3, {7})\text{.}\)

\(f(x) =\)

Answer.

\(1+2x\)

Solution.

We first find the slope \(m\text{:}\)

\begin{equation*} m = \frac{{7} - {-7}}{3 + 4} = \frac{{14}}{{7}}=2 \end{equation*}

Now that we have the slope, we know the linear function has formula \(f(x) = b + 2 x,\) where \(b\) is the vertical intercept. To find the value of \(b\text{,}\) we can plug in either point. Weâ€™ll plug in the first point:

\begin{equation*} {-7} = b + 2 ( -4) \end{equation*}
\begin{equation*} {-7} - 2 (-4) = b \end{equation*}
\begin{equation*} 1 = b \end{equation*}

So the formula is \(f(x)={1+2x}\text{.}\)

Checkpoint 1.9.4.

Find the formula of an exponential function whose graph passes through the points \((1, {7.2})\) and \((4, {1.5552})\text{.}\)

\(f(x) =\)

Answer.

\(12\cdot 0.6^{x}\)

Solution.

Since we are trying to find an exponential function, we start with the formula \(f(x)=a(b)^x.\) We find \(b\) using the ratio \(\dfrac{f(4)}{f(1)}\text{:}\)

\begin{equation*} \frac{f(4)}{f(1)} = \frac{a (b)^{4}}{a(b)^1}= \frac{{1.5552}}{{7.2}} \end{equation*}
\begin{equation*} b^{{3}} = \frac{{1.5552}}{{7.2}} \end{equation*}
\begin{equation*} b = \sqrt[{3}]{\frac{{1.5552}}{{7.2}}} \end{equation*}
\begin{equation*} b = 0.6 \end{equation*}

Now we know that \(f(x) = a (0.6)^{x}\text{.}\) To find the value of \(a\text{,}\) we plug the first point into the formula and solve for \(a\text{:}\)

\begin{equation*} {7.2} = a(0.6)^1 \end{equation*}
\begin{equation*} \frac{{7.2}}{0.6} = a \end{equation*}
\begin{equation*} 12 = a \end{equation*}

So the exponential function is \(f(x)={12\cdot 0.6^{x}}\text{.}\)
