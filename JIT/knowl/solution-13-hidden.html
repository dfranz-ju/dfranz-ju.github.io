Since we are trying to find an exponential function, we start with the formula \(f(x)=a(b)^x.\) We find \(b\) using the ratio \(\dfrac{f(4)}{f(1)}\text{:}\)

\begin{equation*} \frac{f(4)}{f(1)} = \frac{a (b)^{4}}{a(b)^1}= \frac{{1.5552}}{{7.2}} \end{equation*}
\begin{equation*} b^{{3}} = \frac{{1.5552}}{{7.2}} \end{equation*}
\begin{equation*} b = \sqrt[{3}]{\frac{{1.5552}}{{7.2}}} \end{equation*}
\begin{equation*} b = 0.6 \end{equation*}

Now we know that \(f(x) = a (0.6)^{x}\text{.}\) To find the value of \(a\text{,}\) we plug the first point into the formula and solve for \(a\text{:}\)

\begin{equation*} {7.2} = a(0.6)^1 \end{equation*}
\begin{equation*} \frac{{7.2}}{0.6} = a \end{equation*}
\begin{equation*} 12 = a \end{equation*}

So the exponential function is \(f(x)={12\cdot 0.6^{x}}\text{.}\)
