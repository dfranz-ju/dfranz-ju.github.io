We observe that there is a constant difference between the function values of \(5\text{:}\)

\begin{equation*} {13} - 8 = 5 \end{equation*}
\begin{equation*} {18} - {13} = 5 \end{equation*}
\begin{equation*} {23} - {18} = 5 \end{equation*}

That means this table could represent a linear function. To find the formula, we note that the slope is \(5\) and the vertical intercept is \(f(0)=8\text{,}\) so the formula is \(f(x)={8+5x}\text{.}\)
