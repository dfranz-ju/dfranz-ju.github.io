There is not a constant difference between function values, so we try multplication instead. After computing \({7}/5=1.4\text{,}\) we check that it is the consistent multiple between function values:

\begin{equation*} {7} * 1.4 = {9.8} \end{equation*}
\begin{equation*} {9.8} * 1.4 = {13.72} \end{equation*}

So we conclude that this table could represent an exponential function. The initial value is \(f(0)=5\) and we found \(b=1.4\text{,}\) so \(f(x)={5\cdot 1.4^{x}}\text{.}\)
