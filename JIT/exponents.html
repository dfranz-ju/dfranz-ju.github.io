Section 1.2 Exponent Rules

Checkpoint 1.2.1.

Simplify the following expression. Write your answer using positive exponents only.

\(\dfrac{x^{8}}{x^{3}}\) = .

Answer.

\(x^{5}\)

Solution.

\(\dfrac{x^{8}}{x^{3}} = x^{8 - 3} = x^5\)

Checkpoint 1.2.2.

Simplify the following expression. Write your answer using positive exponents only.

\(c^{10} \cdot c^{12}\) = .

Answer.

\(c^{22}\)

Solution.

\(c^{10} \cdot c^{12} = c^{10 + 12} = {c^{22}}\)

Checkpoint 1.2.3.

Simplify the following expression. Write your answer using positive exponents only.

\((z^{4})^{2}\) = .

Answer.

\(z^{8}\)

Solution.

\((z^{4})^{2} = z^{4 \cdot 2} = {z^{8}}\)

Checkpoint 1.2.4.

Simplify the following expression. Write your answer using positive exponents only.

\(\dfrac{y^{5}}{y^{10}}\) = .

Answer.

\(\frac{1}{y^{5}}\)

Solution.

\(\dfrac{y^{5}}{y^{10}} = y^{5 - 10} = y^{-5} = \dfrac{1}{y^{5}}\)
