Section 1.7 Inverse Functions

Checkpoint 1.7.1.

Given that \(g(8)=-4\) and \(g(-8)=8\text{,}\) compute the following:

(a) \(g^{-1}(-4) =\)

(b) \(g^{-1}(8) =\)

Answer 1.

\(8\)

Answer 2.

\(-8\)

Solution.

(a) Since \(g(8)=-4\text{,}\) \(g^{-1}(-4) = 8\text{.}\)

(b) Since \(g(-8)=8\text{,}\) \(g^{-1}(8) = -8\text{.}\)

Checkpoint 1.7.2.

Find the inverse of the function \(y=g(t)= \dfrac{1}{7} t - 9\text{.}\)

\(g^{-1}(y) =\)

Answer.

\(7y+63\)

Solution.

We solve \(y=\dfrac{1}{7} t - 9\) for \(t\text{:}\)

\begin{equation*} y = \frac{1}{7} t - 9 \end{equation*}
\begin{equation*} y+9 = \frac{1}{7} t \end{equation*}
\begin{equation*} 7 y + {63} = t \end{equation*}

So \(g^{-1}(y) = {7y+63}\text{.}\)

Checkpoint 1.7.3.

Determine if the functions below are inverses of each other.

\begin{equation*} f(x) = \frac{1}{2} x - 5 \end{equation*}
\begin{equation*} g(x) = {2x+10} \end{equation*}

Answer:

  • yes

  • no

Answer.

\(\text{yes}\)

Solution.

We evaluate \(f(g(x))\) and check if the result is \(x\text{:}\)

\begin{equation*} f(g(x)) = f({2x+10}) = \frac{1}{2}({2x+10}) - 5 = x + 5 -5 = x \text{.} \end{equation*}

Since \(f(g(x))=x\text{,}\) we can say that yes, \(f(x)\) and \(g(x)\) are inverses of each other.
