First decompose the integrand into partial fractions:

\begin{equation*} \frac{4}{x(x-2)} = \frac{A}{x} + \frac{B}{x-2} \end{equation*}

This yields

\begin{equation*} 4 = A(x-2) + Bx \end{equation*}

Plug in \(x=2\) to solve for \(B\) and \(x=0\) to solve for \(A\text{:}\)

\begin{equation*} 4 = 0 + 2B \implies B=2 \end{equation*}
\begin{equation*} 4=A(-2) + 0 \implies A=-2 \end{equation*}

Now perform the integration

\begin{align*} \int \frac{4}{x(x-2)}\, dx \amp = \int\left(\frac{-2}{x} + \frac{2}{x-2}\right) \, dx\\ \amp = -2\ln|x| + 2\ln|x-2| + C \end{align*}
