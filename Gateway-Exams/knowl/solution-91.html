Solution 2.4.3.1.

Use substitution with \(u=\sqrt{x}=x^{1/2}\) and \(du = \dfrac{1}{2\sqrt{x}} \, dx\text{,}\) so \(2 \, du = \dfrac{1}{\sqrt{x}} \, dx\text{:}\)

\begin{align*} \int \frac{\cos(\sqrt{x})}{\sqrt{x}} \, dx \amp = \int 2 \cos(u) \, du\\ \amp = 2\sin(u) + C\\ \amp = 2\sin(\sqrt{x}) + C \end{align*}
