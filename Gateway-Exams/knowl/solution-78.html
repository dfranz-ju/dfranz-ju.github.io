Solution 2.2.6.1.

Use integration by parts with

\begin{align*} u \amp = t \amp dv \amp = \sin(t) \, dt\\ du \amp = dt \amp v \amp = -\cos(t) \end{align*}

Then

\begin{align*} \int t\sin(t) \, dt \amp = t\cdot (-\cos(t)) - \int -\cos(t) \, dt\\ \amp = -t\cos(t)+ \int \cos(t) \, dt\\ \amp = -t\cos(t) + \sin(t) + C \end{align*}
