First decompose the integrand into partial fractions:

\begin{equation*} \frac{3}{(x+2)(x+3)} = \frac{A}{x+2} + \frac{B}{x+3} \end{equation*}

This yields

\begin{equation*} 3 = A(x+3) + B(x+2)\text{.} \end{equation*}

Plug in \(x=-2\) to solve for \(A\) and \(x=-3\) to solve for \(B\text{:}\)

\begin{equation*} 3 = A + 0 \implies A=3 \end{equation*}
\begin{equation*} 3 = 0 - B \implies B=-3 \end{equation*}

Now perform the integration:

\begin{align*} \int \frac{3}{(x+2)(x+3)} \amp = \int\left(\frac{3}{x+2} - \frac{3}{x+3}\right) \, dx\\ \amp = 3\ln|x+2| - 3 \ln | x+3| + C \end{align*}
