Solution 2.1.3.1.

Use substitution with \(u=5z\) and \(du = 5\, dz\text{,}\) so \(\dfrac{1}{5}\, du = dz\text{.}\)

\begin{align*} \int 3e^{5z} \, dz \amp = \int 3e^u\cdot \frac{1}{5} \, du\\ \amp = \frac{3}{5} \int e^u \, du\\ \amp = \frac{3}{5}e^u + C\\ \amp = \frac{3}{5}e^{5z} + C \end{align*}
