Solution 2.4.8.1.

Use substitution with \(u=9+x^3\) and \(du = 3x^2 \, dx\text{,}\) so \(\dfrac{1}{3} \, du = x^2 \, dx\text{:}\)

\begin{align*} \int \frac{2x^2}{(9+x^3)^2} \, dx \amp = \int \frac{2}{3} u^{-2} \, du\\ \amp = \frac{2}{3}\frac{u^{-1}}{-1} + C\\ \amp = -\frac{2}{3}\cdot \frac{1}{9+x^3} + C \end{align*}
